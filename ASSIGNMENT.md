# YEU CAU

Báo cáo: source code có chú thích trong từng hàm trong file cuả project, báo cáo word gồm khảo sát hiện trạng từ đó xác định các thực thể (thuộc tính khóa -danh hiệu, thuộc tính mô tả), xác định mối liên kết, từ đó đưa ra ERD, xem các quan hệ dự tuyển trong ERD có đạt chuẩn 3 thì chuyển thành các bảng trong database MySQL. Rồi các em tạo các entity class nhé, trong entity thì xác định mối liên kết. Xây dựng code theo mô hình Java Swing MVC + Hiebernate/JPA gồm các Entity/DTO/DAO/Services , rồi đến Controller, Views và sau cùng là MainApp.

Có bộ dữ liệu thử cho đủ các trường hợp chạy: file script tạo data, các JPQL đủ cho hệ thống các em cần.

View: ưu tiên cho thiết kế MainForm/SubForm nhé thực hiện CRUD và một số thao tác tìm kiếm thống kê với JPQL.
